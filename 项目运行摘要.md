## 🔧 运行命令

### 启动区块链节点
```bash
cargo build --release
./target/release/node-template --dev
```

### 启动前端应用
```bash
cd frontend
npm install
npm start
```

### 运行测试
```bash
node test_voting.js
node test_verification_simple.js
```

## 🎯 核心特性

1. **去中心化任务管理**：用户可以创建、接受、完成任务
2. **社区验证机制**：任务完成后需要社区投票验证
3. **自动化流程**：达到投票阈值自动完成验证
4. **声誉系统**：基于任务完成和验证参与度计算
5. **现代化界面**：React前端提供友好的用户体验

## 🌟 项目亮点

- **完全去中心化**：无需中心化服务器
- **社区驱动**：任务质量由社区验证
- **透明公开**：所有操作记录在区块链上
- **激励机制**：通过token奖励激励参与
- **可扩展性**：模块化设计便于功能扩展
